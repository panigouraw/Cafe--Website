# ☕ Coffee Shop Website

A modern and responsive coffee shop website built using HTML, CSS, and JavaScript.

## 📌 Project Overview

This project is a front-end coffee shop website designed to showcase coffee products with an attractive user interface and interactive features.

## ✨ Features

- ☕ Beautiful coffee shop landing page
- 📱 Responsive design for mobile and desktop
- 🛒 Add to cart functionality
- 🔔 Buy Now popup message
- 🧾 Product cards with coffee details
- 🎨 Modern and clean UI design
- 📌 Navigation menu

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Run

1. Download or clone this repository
2. Open the project folder
3. Open `index.html` in your browser
